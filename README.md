# Multi-Game-System
Multi Game System is a program where users can play a game by choosing out of three games. Users have three games to play in a single program.

## Overview

This project is a game application designed to play multiple games on one platform. It utilizes Borland technology for graphics. All users can view the game interface to select different game modules. 

## ❇️Features

- Borland using for user graphics interface
- C/C++ programming used for logical algorithms.
- Data Structure and Algorithms.
